# Mini Framework

Welcome to the Mini Framework! This project is a lightweight framework designed to help you build web applications with a focus on simplicity and performance. 

## Features

- **Virtual DOM**: Efficiently updates the UI by minimizing direct DOM manipulations.
- **Component-Based Architecture**: Create reusable components with lifecycle methods.
- **State Management**: Centralized store for managing application state with reactivity.
- **Routing**: Built-in routing system for handling navigation and URL changes.
- **Event Handling**: Custom event system for managing events across components.
